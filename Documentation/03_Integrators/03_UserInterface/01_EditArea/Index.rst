Edit Area
=========

	:Reduce headers:
		Recommended: Reduce headers from seven to four. Proper translation for headers.

	:Improve images:
		Recommended: Enhance the maximum limit of the image property columns (imagecols) from 7 to 12. Enable cutting of images: Move the image properties height and width from integer to string. You can enter values like 100c (c for cut). You have to update your database once. If you are using TYPO3 without a responsive framework, please select [default] for image improvement.

	:Default language icon:
		Set the default language icon to Englisch or German. Recommended, if you have a localised website and the default language is English or German.

	:Big form fields:
		Recommended: TypoScript templates: Display include static templates with place for upto 40 items and an enhanced width.
